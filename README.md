# React Proyecto Prueba

Base reutilizable con:

- React + TypeScript + Vite
- Tailwind CSS
- React Router
- React Hook Form 

## Requisitos

- Node.js 20+
- npm 10+
- Git

## Clonar e iniciar el proyecto

1. Clonar el repositorio:

```bash
git clone <URL_DEL_REPO>
```

2. Entrar a la carpeta del proyecto:

```bash
cd prueba
```

3. Instalar dependencias:

```bash
npm install
```

4. Levantar en desarrollo:

```bash
npm run dev
```

5. Abrir en el navegador la URL mostrada por Vite (normalmente `http://localhost:5173`).

## Configuracion de endpoints

Revisa y ajusta los endpoints en `env-config.ts`:

- `apiUrl`
- `loginData`
- `menuData`
- `contentData`
